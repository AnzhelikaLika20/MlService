# API Endpoints

## Overview

The API provides endpoints for interacting with the application.

## Endpoints

### GET /api/status

Returns the current status of the application.

### POST /api/data

Creates new data in the system.

### GET /api/data/{id}

Retrieves data by ID.

### PUT /api/data/{id}

Updates existing data.

### DELETE /api/data/{id}

Deletes data by ID.
